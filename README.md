# 膳食管理健康系统 python94

## 项目概述

膳食管理健康系统是一款基于Python语言开发的软件，主要用于辅助用户进行健康饮食管理，通过科学的方法帮助用户规划膳食，提高生活质量。

## 技术栈

- **后端**: Python
- **框架**: Flask
- **数据库**: MySQL

## 功能模块

- 用户管理
  - 用户注册与登录
  - 用户信息管理
- 膳食管理
  - 食物热量查询
  - 膳食计划制定
  - 膳食记录统计
- 健康数据分析

## 系统角色

- **普通用户**: 使用系统进行膳食管理和健康数据追踪
- **管理员**: 对系统进行维护，管理用户信息

## 运行环境

- Python 3.8 或以上版本
- MySQL 5.7 或以上版本
- Flask 框架

## 部署步骤

1. 安装Python环境
2. 安装MySQL数据库
3. 克隆项目代码到本地
4. 安装依赖库（使用pip）
5. 配置数据库
6. 运行Flask应用
